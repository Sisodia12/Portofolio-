Projects

College Management Website

Developed a responsive college management website using HTML, CSS, JavaScript, and Tailwind CSS. The platform facilitates user registration, course management, and event scheduling, enhancing communication between students and faculty. Key features include:

  • User Registration & Login: Secure account creation and access for students and faculty.
  • Course Management: Faculty can manage courses, while students can enroll and track progress.
  • Event Scheduling: A calendar feature for viewing and managing upcoming events.
  • Responsive Design: Ensured seamless usability across devices using Tailwind CSS for rapid styling.

Quiz Application

This quiz application allows users to test their knowledge on various topics through interactive quizzes. Users can select a quiz category, answer questions, and receive immediate feedback on their performance. The goal is to provide an engaging way to learn and assess knowledge.

Technologies Used: HTML, CSS, JavaScript.
